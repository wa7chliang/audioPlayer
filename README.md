# audioPlayer
audio function library

### options
```el```: 容器元素
```songList```: 列表(string || object{url: string})
```mutex```: 是否阻止多个播放器同时播放(bool)

### API
```play()```: 播放
```pause()```: 暂停
```toggle()```: 切换
```next()```: 下一首
```prev()```: 上一首
```getBufferedPercent()```: 获取已缓冲百分比
```getCurrentTimePercent()```: 获取当前播放百分比
